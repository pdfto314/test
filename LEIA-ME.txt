Jogatina Soundboard (versão estável)

✅ Objetivo (como era no início):
- GitHub Pages
- iPad friendly (botão para liberar áudio)
- Lista automaticamente os áudios da pasta /audio usando GitHub API
- Cada subpasta em /audio vira um Tema
- Ambiente (1 canal em loop) + Efeitos (multi-camadas)
- Sets (cenas) salvos no navegador (localStorage)

Estrutura recomendada:
audio/
  Floresta/
    rain.mp3
    birds.mp3
  Taverna/
    tavern_loop.mp3
  Batalha/
    sword_hit.mp3
    crowd.mp3

Configurar:
- Abra app.js e ajuste:
  OWNER, REPO, BRANCH, AUDIO_PATH

Publicar:
- Commit + Push (GitHub Pages aponta para branch main / root)
