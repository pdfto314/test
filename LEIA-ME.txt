Como usar (Windows):

1) Extraia estes arquivos dentro da pasta do seu projeto (test/), onde está o index.html.
   Deve ficar assim:
   test/
     index.html
     app.js
     style.css
     audio/
       temas/ (se você usa esse formato)
     gerar_playlist.py
     generate_playlist.bat

2) Rode o BAT:
   generate_playlist.bat

Se der erro de 'py não encontrado':
- Instale Python 3 do site oficial OU Microsoft Store
- E marque 'Add to PATH' (ou use o launcher 'py' do Windows)

Depois: Commit + Push no GitHub Desktop.
